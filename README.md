## Getting Started

1. Clone the repository
```bash
git clone https://github.com/ShahzaibaslamIT/portfolio.git
cd portfolio
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm run dev
```

4. Open [http://localhost:5000](http://localhost:5000) in your browser
